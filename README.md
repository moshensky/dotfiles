## Install

```sh
git clone https://github.com/moshensky/dotfiles ~/dotfiles
~/dotfiles/scripts/setup.sh
```

### Install tmux plugins

Start tmux session:

```sh
tmux
```

Fetch plugins and source them press: `<c-b> + I`
To source .tmux.conf: `<c-b> + R`
